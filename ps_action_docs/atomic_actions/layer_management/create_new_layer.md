# Create New Layer

**Action:** `make`  
**Target:** `layer`

Creates a new, empty raster layer.

## Create Empty Layer

```json
{
    "_obj": "make",
    "_target": [
        {
            "_ref": "layer"
        }
    ],
    "layerID": 473
}
```

## Create and Name Layer

Creates a layer and names it immediately.

```json
{
    "_obj": "make",
    "_target": [
        {
            "_ref": "layer"
        }
    ],
    "using": {
        "_obj": "layer",
        "name": "My Layer Name"
    }
}
```

## Parameters

- `layerID`: (Optional) Internal ID for the new layer. Usually generated by Photoshop, but can be ignored when creating new actions.
- `using`: (Optional) Object defining properties of the new layer, such as `name`.

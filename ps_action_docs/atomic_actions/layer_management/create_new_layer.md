# Create New Layer

**Action:** `make`  
**Target:** `layer`

Creates a new, empty raster layer.

## JSON Structure

```json
{
    "_obj": "make",
    "_target": [
        {
            "_ref": "layer"
        }
    ],
    "layerID": 473
}
```

## Parameters

- `layerID`: (Optional) Internal ID for the new layer. Usually generated by Photoshop, but can be ignored when creating new actions.

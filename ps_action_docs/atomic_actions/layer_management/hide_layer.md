# Hide Layer

**Action:** `hide`
**Target:** `layer`

Hides the specified layer.

## JSON Structure

```json
{
    "_obj": "hide",
    "null": [
        {
            "_name": "Fog",
            "_ref": "layer"
        }
    ]
}
```

## Parameters

- `null`: (Array) Target to hide.
    - `_ref`: `layer`.
    - `_name`: Name of the layer to hide (or use ordinal target).

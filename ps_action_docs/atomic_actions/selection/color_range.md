# Color Range

**Action:** `colorRange`
**Target:** Selection (implicitly)

Creates a selection based on a specific color range.

## JSON Structure

```json
{
    "_obj": "colorRange",
    "colorModel": 1,
    "colors": {
        "_enum": "colors",
        "_value": "midtones"
    },
    "fuzziness": 40,
    "minimum": {
        "_enum": "colorRangeMinimum",
        "_value": "minimumNone"
    },
    "maximum": {
        "_enum": "colorRangeMaximum",
        "_value": "maximumNone"
    }
}
```

> Note: `raindrop.json` uses a complex set of Gaussian parameters (`posGaussParams`, etc.) which are auto-generated by the command. For general use, selecting `midtones`, `shadows`, or `highlights` is common.

## Parameters

- `colors`: (Enum) The color range to select.
    - `_value`: `midtones`, `shadows`, `highlights`, `reds`, etc.
- `fuzziness`: (Integer) Fuzziness value (0-200).
- `colorModel`: (Integer) Color model ID.

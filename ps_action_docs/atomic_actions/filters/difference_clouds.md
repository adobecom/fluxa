# Difference Clouds

**Action:** `differenceClouds`
**Target:** None (implicitly targets active layer)

Applies the Difference Clouds filter, using the foreground and background colors to generate a cloud pattern that is blended with the existing image using the Difference mode.

## JSON Structure

```json
{
    "_obj": "differenceClouds"
}
```

## Parameters

None. This action takes no parameters.
